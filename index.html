Oops! Votre navigateur est mal configuré!


Pour naviguer sur l'intranet et l'internet, vous devez configurer votre navigateur avec la détection automatique du proxy ou bien avec le script de configuration PAC: http://www.dsi.univ-nantes.fr/cache.pac

Instructions de paramétrage pour les principaux navigateurs:
Cochez la case "Détection automatique des paramètres de proxy pour ce réseau" dans le menu Outils → Options → Avancé → Réseau → Paramètres

Cochez la case "Détecter automatiquement les paramètres de connexion" dans le menu Outils → Options Internet → Connexions → Paramètres réseau → Détecter automatiquement les paramètres de connexion

Saisir http://www.dsi.univ-nantes.fr/cache.pac comme URL de fichier PAC dans le menu Safari → Préférences → Avancées → Proxies: Modifier les réglages → Utiliser un fichier PAC

Pour configurer manuellement une application ne supportant pas la détection automatique, veuillez utiliser les paramètres suivants:

Serveur: cache.wifi.univ-nantes.fr   Port: 3128
